This is a guide for the most common situations:

Situation: There is a product bug
Recommendation:
- Product: Add some URL with the bug filed - Automation: Any option is fine since the product is failing. Next Run: Triage will be automatic if the automation issue is the same.
Situation: Facing deprecated test.
Recommendation:
  • Product: get sure that the product is working fine and choose it.
  • Automation: Won't Fix.
Next Run: Triage will be automatic if the automation issue is the same.
Situation: Test is failing but it is not yours.
Recommendation: Make the triage, then go to Application Issues and assign the ticket to the current owner.
